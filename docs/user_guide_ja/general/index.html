はじめよう

どんなソフトウェアアプリケーションも学習する努力が必要です。学ぶ過程をできるだけ楽しくして、学習曲線を最小化するようベストを尽くしましょう。

最初のステップは、CodeIgniterをインストールすることです。インストールが終わったら、目次にあるイントロダクションのトピックを全部読んでください。

次に 一般的になトピック のページを順に読みます。各トピックは、前のトピックを前提に話を進めています。そこに書いてあるサンプルコードはぜひ動かして試してみてください。

いったん基本を理解したなら、組み込みのライブラリとヘルパ関数の使い方を知るために クラスリファレンス と ヘルパ関数リファレンス のページを調べていく準備が整ったことになります。

質問や問題があれば、コミュニティフォーラム をお気軽にご利用ください。他の人が投稿したサンプルコードを見たければ Wiki を見てみてください。
