『CodeIgniter徹底入門』サンプルプログラム

2009/06/10  Rev 1.4


この度は『CodeIgniter徹底入門』をお買い上げいただき、誠にありがとうございます。
このパッケージは、『CodeIgniter徹底入門』のサンプルプログラムです。


■サンプルプログラムのインストール方法
このサンプルプログラムは、以下のURL からダウンロードできます。

http://codeigniter.jp/tettei/

上記の URL からダウンロードしたサンプルプログラムファイルを解凍すると、章別の
ディレクトリが作成されます。

章別のディレクトリの中身を、

 * Windows 環境   --->「C:¥xampp¥htdocs¥CodeIgniter」ディレクトリ
 * GNU/Linux 環境 --->「/opt/lampp/htdocs/CodeIgniter」ディレクトリ

にコピーしてください。

dot.htaccess は「.htaccess」のサンプルです。

GNU/Linux 環境では、必要な場合は、本文の指示に従って、ディレクトリの
パーミッション設定を行ないます。
Windows 環境では特に必要ありません。

この状態で、ブラウザから、インストールしたサンプルプログラムにアクセスできる
ようになります。

なお、『CodeIgniter徹底入門』では、CodeIgniter の開発環境として Eclipse を使用
しています。Eclipse からは、インストールされた新しいファイルが見えませんので、
Eclipse を起動し、PHPエクスプローラーで「CodeIgniter」プロジェクトを右クリック
して［更新］を選択します。

これで、Eclipse からもインストールされた新しいファイルが見えるようになります。

以上で、サンプルプログラムのインストールは完了です。
